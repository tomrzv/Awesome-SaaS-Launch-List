## Submission Tips

Before submitting your product, prepare:

- a short one-line pitch
- a longer product description
- logo and screenshots
- launch tagline
- category selection
- backlink / website URL
- founder profile
- social links
- promo code or special offer if relevant

### Best practices

- Tailor each description to the platform
- Avoid copy-pasting the same pitch everywhere
- Use screenshots with real UI
- Add your best category keywords naturally
- Reply to comments when the platform supports discussion
- Track live backlinks and referral traffic after submission

## 🤝 Contributing

Contributions are welcome.

To add a new platform:

1. Fork the repository
2. Add the platform in the most relevant section
3. Include the name, URL, and a concise description
4. Make sure it is not already listed
5. Open a pull request

### Suggested inclusion rules

- The platform should be active
- It should be useful for launching, promoting, reviewing, or discovering products
- Directories with clear category relevance are preferred
- Dead, abandoned, or spam-heavy sites should be avoided
